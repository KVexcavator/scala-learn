Чистая функция
• Возвращает единственное значение.
• Вычисляет результат, опираясь только на свои аргументы.
• Не изменяет существующие значения.
==========================================
Неизменяемые значения
Состояние — это экземпляр значения, хранящийся в одном месте и доступный из кода. Если это значение можно изменить, то, следовательно, мы имеем изменяемое состояние. Кроме того, если к этому изменяемому состоянию можно получить доступ из разных частей
кода, это общее изменяемое состояние.
-------------------------------------------
В Java экземпляры List и ArrayList изменяемые, String неизменяемый.
В Scala встроены неизменяемые коллекции, включая списки.
Scala использует тот же тип String, что и Java.
Список List в Scala — неизменяемый. Все операции с ним возвращают новый список
