Scala 3, пример SBT-зависимостей для Akka:
libraryDependencies ++= Seq(
  "com.typesafe.akka" %% "akka-actor-typed" % "2.6.19", // Для typed API
  "com.typesafe.akka" %% "akka-actor" % "2.6.19"        // Для классического API
)
----------------------------------------------------------
Mногие параллельные вычисления, могут продолжаться после завершения основной программы. Чтобы гарантировать завершение, добавить в файл build.sbt:
fork := false
